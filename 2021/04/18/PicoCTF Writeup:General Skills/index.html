PicoCTF Writeup - General Skills

发表于 2021-04-18

PicoCTF是由卡耐基梅隆大学创办的CTF赛事，由于其主要受众是高中生，所以题目难度偏低，比较适合新手入门。除了部分二进制题目外，flag格式为picoCTF{xxx}。

General Skills

  • Obedient Cat | 5 points

This file has a flag in plain sight (aka “in-the-clear”). Download flag.

这个文件里有明文的flag。

flag:picoCTF{s4n1ty_v3r1f13d_2fd6ed29}

  • Python Wrangling | 10 point

Python scripts are invoked kind of like programs in the Terminal… Can you run this Python script using this password to get the flag?

Python脚本可以像程序一样在终端中被调用，你能用此密码运行这个Python脚本来获取flag吗？

flag:picoCTF{4p0110_1n_7h3_h0us3_ac9bd0ff}

  • Wave a flag | 10 points

Can you invoke help flags for a tool or binary? This program has extraordinarily helpful information…

你能调用工具或二进制文件的帮助标志吗？这个程序具有非常有用的信息…

flag:picoCTF{b1scu1ts_4nd_gr4vy_f0668f62}

  • Nice netcat… | 15 points

There is a nice program that you can talk to by using this command in a shell: $ nc mercury.picoctf.net 21135, but it doesn’t speak English…

这有一个不错的程序，你可以在Shell中使用以下命令与之交谈：$ ncmercury.picoctf.net 21135，但是它不会说英语…

nc号称网络安全界的瑞士军刀，用法很多，可以参考国光小姐姐的这篇教程：nc命令教程。直接执行题中命令的话会输出一串ascii编码的字符，解码的方法有很多，我这里用的是awk。

flag:picoCTF{g00d_k1tty!_n1c3_k1tty!_afd5fda4}

  • Static ain’t always noise | 20 points

Can you look at the data in this binary: static? This BASH script might help!

你能看一下以下二进制文件中的数据吗，这个BASH脚本可能会有所帮助！

flag:picoCTF{d15a5m_t34s3r_6f8c8200}

  • Tab, Tab, Attack | 20 points

Using tabcomplete in the Terminal will add years to your life, esp. when dealing with long rambling directory structures and filenames: Addadshashanammu.zip

在终端中使用自动补全会延长你的寿命，尤其是处理冗长的目录结构和文件名时：Addadshashanammu.zip

flag:picoCTF{l3v3l_up!_t4k3_4_r35t!_2bcfb2ab}

  • Magikarp Ground Mission | 30 points

Do you know how to move between directories and read files in the shell? Start the container, ssh to it, and then ls once connected to begin. Login via ssh as ctf-player with the password, a13b7f9d

你知道如何在目录之间移动并读取Shell中的文件吗？启动容器，将其“ssh”启动，在连接后执行“ls”。通过ssh作为ctf-player登录，密码为a13b7f9d。

这里点击题目右侧的Start Instance之后稍等片刻会给出连接到远程实例的ssh命令。连接后，主目录下会有flag片段以及下一个flag片段的线索。

flag:picoCTF{xxsh_0ut_0f_\/\/4t3r_71be5264}

  • Lets Warm Up | 50 points

If I told you a word started with 0x70 in hexadecimal, what would it start with in ASCII?

如果我告诉你一个以十六进制0x70开头的单词，它在ASCII编码中将以什么开头？

flag:picoCTF{p}

  • Warmed Up | 50 points

What is 0x3D (base 16) in decimal (base 10)?

十六进制数0x3D在十进制中是多少？

flag:picoCTF{61}

  • 2Warm | 50 points

Can you convert the number 42 (base 10) to binary (base 2)?

你能否将数字42转换为二进制数？

flag:picoCTF{101010}

  • what’s a net cat? | 100 points

Using netcat (nc) is going to be pretty important. Can you connect to jupiter.challenges.picoctf.org at port 41120 to get the flag?

使用netcat（nc）将非常重要。你可以连接到jupiter.challenges.picoctf.org的端口41120来获取flag吗？

flag:picoCTF{nEtCat_Mast3ry_3214be47}

  • strings it | 100 points

Can you find the flag in file without running it?

你可以在不运行文件的情况下找到flag吗？

flag:picoCTF{5tRIng5_1T_7f766a23}

  • Bases | 100 points

What does this bDNhcm5fdGgzX3IwcDM1 mean? I think it has something to do with bases.

bDNhcm5fdGgzX3IwcDM1是什么意思？我认为这与base编码有关。

flag:picoCTF{l3arn_th3_r0p35}

  • First Grep | 100 points

Can you find the flag in file? This would be really tedious to look through manually, something tells me there is a better way.

你可以在文件中找到flag吗？手动浏览确实很乏味，直觉告诉我有更好的方法。

flag:picoCTF{grep_is_good_to_find_things_5af9d829}

  • Based | 200 points

To get truly 1337, you must understand different data encodings, such as hexadecimal or binary. Can you get the flag from this program to prove you are on the way to becoming 1337? Connect with nc jupiter.challenges.picoctf.org 29956.

要真正成为1337，你必须了解不同的数据编码，例如十六进制或二进制。你可以从该程序中获得flag来证明你即将成为1337吗？连接到nc jupiter.challenges.picoctf.org的29956端口。

连接后是一个交互式程序，需要在45秒内将给出的字符串转换为英文字符并输入，一共三道题（第一题答案已经在输出里给出，ctrl+v即可）。我对自己手速比较自信，直接用Python Shell完成的。

与此同时Python Shell中的交互内容：

flag:picoCTF{learning_about_converting_values_b375bb16}

  • plumbing | 200 points

Sometimes you need to handle process data outside of a file. Can you find a way to keep the output from this program and search for the flag? Connect to jupiter.challenges.picoctf.org 4427.

有时你需要处理文件正在输出的数据。你能找到一种方法来保留此程序的输出并搜索flag吗？连接到jupiter.challenges.picoctf.org 4427。

flag:picoCTF{digital_plumb3r_5ea1fbd7}

  • mus1c | 300 points

I wrote you a song. Put it in the picoCTF{} flag format.

我给你写了一首歌。将其放入picoCTF {}的flag格式中。

这里参考提示：

Do you think you can master rockstar?

google一下发现这是居然是个编程语言，项目地址：RockstarLang/rockstar。在项目主页上能看到将rockstar源码转换为其他语言的工具。亲测发现Python语言的版本存在bug，Ruby语言的版本运行良好。先进行工具的安装：

根据项目说明运行rockstar源码，并将输出内容进行ascii解码：

flag:picoCTF{rrrocknrn0113r}

  • flag_shop | 300 points

There’s a flag shop selling stuff, can you buy a flag? Source. Connect with nc jupiter.challenges.picoctf.org 9745.

这有一家卖flag的商店，你能买到flag吗？用nc jupiter.challenges.picoctf.org 9745来连接。

连接上去是一个购买flag的交互式程序，但是初始资金是不够买我们需要的flag的。下载源码进行审计，发现38行计算花销时没使用unsigned int，想到可以通过购买非常大数量的flag将其溢出成负数从而增加余额。

交互过程：

flag:picoCTF{m0n3y_bag5_65d67a74}

  • 1_wanna_b3_a_r0ck5tar | 350 points

I wrote you another song. Put the flag in the picoCTF{} flag format

我给你写了另一首歌。将其放入picoCTF {}的flag格式中。

同mus1c那一题，不过这次不能直接运行，而要转成Ruby代码后在其中寻找flag。

将其中出现的数字进行ascii解码：

flag:picoCTF{BONJOVI}
